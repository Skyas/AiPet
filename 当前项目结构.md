## 项目结构

```
AiPet/
├── backend/                        # Python FastAPI 后端
│   ├── main.py                     # 应用入口 & 模块初始化
│   ├── version.py                  # 版本号
│   ├── requirements.txt            # 核心依赖
│   ├── requirements_voice.txt      # 语音模块依赖（按需安装）
│   ├── api/
│   │   ├── chat.py                 # 对话路由（SSE 流式 + Vision 注入）
│   │   ├── screen.py               # 截图 / Vision 分析 / 主动引擎控制
│   │   ├── settings_api.py         # 配置 CRUD + 热重载 + 模块开关
│   │   └── prompts_api.py          # Prompt 卡片 CRUD
│   ├── config/
│   │   ├── settings.py             # 配置读写 & deep merge
│   │   └── user_config.json        # 用户配置文件（自动生成）
│   └── modules/
│       ├── prompt_manager.py       # Prompt 卡片管理
│       ├── core/
│       │   ├── ai_chat.py          # 文本对话（OpenAI 兼容）
│       │   ├── ai_vision.py        # 视觉路由（multimodal / custom / default）
│       │   ├── screen_capture.py   # mss 截图 + Pillow 压缩
│       │   ├── history.py          # 对话历史（SQLite）
│       │   ├── vision_memory.py    # 屏幕滚动记忆（Jaccard 变化检测）
│       │   └── proactive_engine.py # AI 主动互动引擎（[SILENT] 机制）
│       ├── voice/
│       │   └── __init__.py         # 语音模块占位（Phase 3）
│       └── qq/
│           └── __init__.py         # QQ 模块占位（Phase 4）
│
├── frontend/                       # Electron + Vue 3 前端
│   ├── electron/
│   │   ├── main.js                 # 主进程（窗口 / 托盘 / 快捷键 / IPC）
│   │   └── preload.js              # IPC 桥（contextBridge）
│   ├── src/
│   │   ├── main.js                 # Vue 应用入口
│   │   ├── App.vue                 # 根组件
│   │   ├── router.js               # 路由配置（4 条路由）
│   │   ├── views/
│   │   │   ├── ChatPanel.vue       # 主对话界面（流式 + 主动消息气泡）
│   │   │   ├── VisionPanel.vue     # 游戏陪玩界面（观察流 + 主动模式）
│   │   │   ├── SettingsPanel.vue   # 设置面板（Vision 路由指示器 + 热重载）
│   │   │   └── PromptEditor.vue    # Prompt 角色卡编辑器
│   │   ├── stores/
│   │   │   ├── chat.js             # 对话状态（Pinia）
│   │   │   └── settings.js         # 配置状态（Pinia）
│   │   └── utils/
│   │       ├── api.js              # HTTP / SSE 全部 API 封装
│   │       └── socket.js           # Socket.IO 全局单例
│   ├── index.html
│   ├── vite.config.js              # Vite 配置 + 开发代理
│   ├── package.json
│   └── .npmrc                      # npmmirror 镜像加速
│
├── prompts/                        # Prompt 角色卡（.json）
├── data/                           # 运行时数据库（.db）
└── scripts/                        # Windows 启动 / 安装脚本
```
